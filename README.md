## Cara Instalasi
Untuk menjalankan proyek ini di komputer lokal:
1. Clone repository ini: `git clone https://github.com/Riyancil123/absensi-mahasiswa.git`
2. Masuk ke direktori: `cd absensi-mahasiswa`
3. Install dependencies: `composer install`
4. Salin file env: `cp .env.example .env`
5. Generate key: `php artisan key:generate`
6. Konfigurasi database di file `.env`
7. Jalankan migrasi: `php artisan migrate`
8. Jalankan server: `php artisan serve`
